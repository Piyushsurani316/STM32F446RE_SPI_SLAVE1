# 📡 SPI Protocol Explanation

SPI (Serial Peripheral Interface) is a **synchronous full-duplex communication protocol**.

### Signals Used:

| Signal | Description |
|--------|------------|
| SCK | Serial Clock (Generated by Master) |
| MOSI | Master Out Slave In |
| MISO | Master In Slave Out |
| NSS | Slave Select |

---

### Communication Flow

1. Master pulls NSS LOW.
2. Master generates SCK.
3. Data shifts out on MOSI.
4. Slave samples data on clock edge.
5. After transmission, NSS goes HIGH.

---


If the received data matches exactly, the slave:

✅ Turns ON LED (PA9)  
✅ Waits 50ms  
✅ Turns OFF LED
